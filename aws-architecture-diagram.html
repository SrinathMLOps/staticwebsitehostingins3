📋 Implementation Steps

1 Create S3 Bucket - Name: srinathkaithoju.com, Region: eu-west-2, Disable block public access
2 Upload Website Files - Upload index.html, CSS, JS, images folders
3 Add Bucket Policy - Allow public read access (s3:GetObject)
4 Enable Static Hosting - Set index.html as index document
5 Request SSL Certificate - Use ACM in us-east-1 for *.srinathkaithoju.com
6 Create CloudFront Distribution - Origin: S3 bucket, Enable HTTPS redirect, Attach SSL
7 Update DNS Records - Add CNAME records pointing to CloudFront distribution
